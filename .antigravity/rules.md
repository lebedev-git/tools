# Antigravity Rules

- **Текстовые запросы (ИИ-аналитика)**:
  * ВСЕГДА должны отправляться на провайдер Qwen через эндпоинт `https://qwen.aikit.club/v1`.
  * ВСЕГДА должны использовать модель `qwen3.7-max`.
  * Ни в коем случае НЕ МЕНЯТЬ эту модель и эндпоинт для текстовой аналитики.

- **Генерация изображений (Инфографика)**:
  * Запросы на генерацию картинок отправляются на Codex Sale через эндпоинт `https://codex.sale/v1`.
  * Используется модель `gpt-image-2`.
  * Параметры логотипа (`logo`) и общей фотографии (`photo`) должны передаваться только на генерацию изображения.
